☁️ Sincronización entre dispositivos

Sincroniza tu progreso entre celular y computadora gratis con Firebase de Google. Solo necesitas una cuenta de Google.

1
Crear proyecto Firebase
Ve a console.firebase.google.com → Crear un proyecto → ponle un nombre (ej: life-os) → Continuar sin Analytics → Crear
2
Activar inicio de sesión con Google
En el menú izquierdo: Build → Authentication → Comenzar → Sign-in method → Google → Habilitar → selecciona tu correo de soporte → Guardar
3
Crear base de datos Firestore
Build → Firestore Database → Crear base de datos → Iniciar en modo de prueba → Siguiente → Habilitar
4
Registrar tu app web
⚙️ Configuración del proyecto → Tus apps → icono Web (</>) → Registra la app (nombre: life-os) → Copia el objeto firebaseConfig
5
Agregar dominio autorizado
Authentication → Settings → Dominios autorizados → Agregar dominio → escribe: clinicamedicageneral.github.io → Agregar
